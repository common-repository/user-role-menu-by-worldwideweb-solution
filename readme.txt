======== DESCRIPTION ========
User Role Menu for WordPress is a great plugin, which completely integrates with the native menu system. You will be able to easily create a public menu, a menu for user when they are logged in and menus for each User Role. 


== INSTALLATION ==

1. Upload the 'modal-login folder to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. In the menu you will find Appearance > Menus
